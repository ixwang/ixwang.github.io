How to deploy a new post

In terminal, go to the folder of your hexo,

hexo new post “articletitle” //create a new article

go to hexo/source/_posts/articletitle.md to edit your post

hexo g // or hexo generate to generate the static web

hexo d // or hexo deploy to deploy the local static web to server

done.

How to change theme

Download your theme file, put the folder into hexo/themes,

edit _config.yml in the root directory, find keyword ‘theme:’, set your theme name there.

excute hexo g, hexo d will change the theme.
